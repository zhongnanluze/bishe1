项目简介

为校园生活注入
AI 智能动力

"文泽奇妙小AI"是一款面向高校学生的智能服务系统，采用前后端分离架构，基于 Vue 3 和 FastAPI 构建。系统集成了 DeepSeek 大语言模型，通过多智能体协作，为学生提供学生事务办理和学业管理的智能化服务。

核心功能

全方位的 智能服务

系统覆盖学生在校期间的各类事务需求，通过 AI 智能体提供精准、高效的服务。

01
💬

智能多轮对话

基于 DeepSeek API 的流式对话，支持上下文理解和多轮交互，打字机效果让体验更自然。

02
📋

学生事务办理

证件补办、学费缴纳、饭卡充值、办事流程查询，一站式解决校园事务需求。

03
🎓

学业管理服务

选课退课、课表查询、成绩查询、GPA 计算、学业日历，全面覆盖学业场景。

04
🤖

多智能体路由

智能路由系统自动识别用户意图，将请求分配给事务智能体或学业智能体处理。

05
🔐

安全认证系统

JWT Token 认证机制，支持用户注册登录、密码修改、头像上传和个人信息管理。

06
📚

知识库维护

管理员可维护知识库内容，为 AI 智能体提供最新、最准确的校园信息服务。

07
📝

会话历史管理

自动保存对话历史，支持会话列表查看、切换和删除，聊天记录持久化存储。

08
🌙

炫酷暗黑主题

精心设计的暗黑模式界面，配合 Three.js 特效和毛玻璃效果，视觉体验极佳。

09
📱

响应式设计

完美适配桌面端和移动端，自适应布局确保在任何设备上都有良好的使用体验。
